Download our game!

Thanks to ElectronJS , a framework for building desktop applications using Javascript, HTML, and CSS, we are able to port the game so you can try it out! Now you can experience Fry's courageous action on the go, where ever you want. The game doesn't need any internet connection and has very minimal assets so almost any computer can play it. It is in a zip file, so all you need to do is extract it and then you can start playing.

What are you waiting for? Download and play!
